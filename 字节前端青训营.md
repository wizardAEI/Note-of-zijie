## HTML



### 页面整体

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width,initial-scale=1.0"><!--用于移动端，定义设备屏幕上用来显示网页的区域-->
    <meta name="keywords" content="html">
    <meta name="description" content="这是页面的描述，如果存在，则会在搜索引擎的结果中显示"> 
    <title>title</title>
  </head>
  <body>
   <p>Hello World</p>
  </body>
</html>

    
<!--
HTML包含多个THML元素，元素具备不同的特性
HTML标签不区分大小写
元素可嵌套，可以包括属性的额外信息
-->
```

```
<!DOCTYPE html>   放在HTML页面前面的位置，加上之后就会按照W3C的HTML5标准渲染页面
<html>    页面根元素
<head>    对用户不可见，其中包含例如面向搜索引擎的关键字，页面描述，字符编码说明，CSS样式
<body>    包含用户可以看到的文本


例子：
<meta>  元数据  
 <meta name="keywords" content="html">  关键字，用于搜索引擎的关键字
＜meta http-equiv="expires" content="Wed, 20 Jun 2021 22:33:00 GMT"＞  可以用于设定网页的到期时间。一旦网页过期，必须到服务器上重新传输。 
```

script：

```js
<script type="text/javascript" src="index.js" defer>  异步下载脚本，页面完全加载后开始加载脚本
```



### 常用元素
常用元素:内联元素，块级元素，行内块级元素



### 语义化

为了让开发者阅读，标签采用语义化的写法

例如p标签和div标签，虽然都是块级元素，但是写段落时，也会用p标签

#### 语义化-区块

`<header>...</header>`

展现介绍性的信息，导航，标题，logo，搜索框，作者名称等

`<nav>...</nav>`

提供导航链接，如菜单，目录，索引等

`<article>`

独立的文档，页面，应用，帖子

`<section>`

按主题将内容隔开，内含标题

`<aside>`

 与整体没太大关系的部分或文章目录，例如广告，tips，引用内容等，常在侧边栏

`<footer>`

表示章节页脚，包含该章节作者，版权数据或文档链接等信息



#### 语义化-分组

`<figure>...</figure>`

包裹独立内容如图标，插图，代码块等。`<figcaption>标题/说明<figcaption>`写在内容的最后

`<blockquote>`

块级引用元素

`<dl> <dt> <dd>`

用于描述键值对，用于元数据，属于定义等场景



```html
 <dl>
 <dt>Firefox</dt>
 <dt>Mozilla Firefox</dt>
 <dt>Fx</dt>
 <dd>A free, open source, cross-platform, graphical web browser
   developed by the Mozilla Corporation and hundreds of volunteers.</dd>
</dl> 
```



#### 语义化-文本类

`<cite>` 

引用作品标题

`<time datetime="2001-5-15 19:00">May 15</time>`

引用机器可读的时间日期，datetime为此元素关联的日期，会被解析

`<address>`联系信息

`<mark>`表示需要注意的部分

`<code>`代码

`<small>`免责声明，注意事项



### 多媒体元素

#### `<img />`

- src  图片路径
- alt  对图片的文本描述
- decoding = "async" 异步
- loading="lazy" 懒加载

#### `<picture />`

- 通过`<source>`和一个`<img>`为不同的显示/设备场景提供图像版本
- media属性:依据当前环境下支持类型渲染相应的图片，类似媒体查询
- type属性：MIME类型

例：

```html
<picture>
	<source srcSet="xxx.webp" type="image/webp">
	<img decoding="async" loading="lazy" src="xxx.jpeg">
</picture>
```

#### `<video>/<audio>` 音视频

- src属性，嵌入视频的路径
- controls 是否显示控件或自定义控件
- autoplay 是否自动播放
- source 表示视频的代替资源（会在音视频读取失败后尝试此属性）

### HTML解析（dom树）

字节流->DOM树->样式计算生成CSSOM树->渲染引擎生成Render树->布局计算并绘制到页面

可以通过 box-sizing改变

#### flex 

- flex-direction: row|row-reverse|colum|colum-reverse
- justify-content: flex-start | flex-end | center | space-between
- align-items:  flex-start | flex-end | center | baseline(以中间线为对齐线) | stretch(铺满)



#### cursor属性，改变鼠标样式

例：

```

	cursor:wait
}//指向段落时段鼠标变成缓冲的形状
```
